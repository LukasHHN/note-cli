note-cli

Ein blitzschnelles, leichtgewichtiges Bash-Tool, um Notizen direkt in deinem Terminal zu speichern, zu verwalten und aufzulisten.

🚀 So einfach geht's:

$ ./note.sh add "Meeting um 14 Uhr"
> Notiz gespeichert! Hash: 8f4e2...

$ ./note.sh list
> 8f4e2... - Meeting um 14 Uhr

Besuche das GitHub Repository für die Installation und Dokumentation.
